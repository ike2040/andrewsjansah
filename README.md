# Andrews Jansah - Personal Website

Welcome to the official repository for **Andrews Jansah's Personal Website**. This project showcases a clean, modern, and professional web presence designed to highlight services, products, and professional background.

## 🚀 Features

- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Service Offerings**: A dedicated page showcasing the professional services provided.
- **Product Catalog**: An interactive product listing with detailed descriptions.
- **Contact Form**: An easy way for visitors to get in touch.
- **Dynamic Content**: JavaScript-driven interactive elements to enhance user experience.

## 📁 Project Structure

```bash
├── index.html          # Main landing page
├── services.html       # Services provided
├── products.html       # Product showcase
├── contact.html        # Contact information and form
├── styles.css          # Core design and styling
├── script.js           # Interactive functionality
├── images/             # Visual assets and graphics
└── resources/          # Additional JSON data and resource files
```

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Modern styling with custom properties and responsive layouts.
- **JavaScript**: Dynamic interactivity and data fetching.
- **JSON**: Data storage for products and resources.

## 💾 Installation & Setup

To run this project locally, simply clone the repository and open `index.html` in your web browser:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ike2040/andrewsjansah.git
    ```
2.  **Open in Browser**:
    Double-click `index.html` or use a local development server (e.g., Live Server extension in VS Code).
